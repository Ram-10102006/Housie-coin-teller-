# Bingo Number Generator

A simple web app to generate Bingo numbers from 1 to 90.

## Features

- Shows numbers 1-90 in a grid.
- Highlights numbers that have been drawn.
- Voice announces numbers in English or Telugu.
- Auto-generate numbers every 4 seconds.
- Shows previous number and allows going back.
- Reset button to start a new game.
